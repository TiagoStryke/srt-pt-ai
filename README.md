# Tradutor de Legendas SRT para Português Brasileiro

Uma ferramenta poderosa para traduzir arquivos de legendas SRT para Português Brasileiro com qualidade profissional, utilizando Inteligência Artificial ✨

Diga adeus às legendas geradas automaticamente de baixa qualidade e ofereça uma experiência premium com traduções precisas e naturais! 🎉

## Visão Geral

O Tradutor de Legendas SRT utiliza o poder da IA do Google Gemini para fornecer traduções precisas e naturais para arquivos de legendas SRT, otimizadas especificamente para o português brasileiro. Esta ferramenta fácil de usar garante que os espectadores possam desfrutar de conteúdo de vídeo sem a frustração de legendas mal traduzidas.

## Como Começar

Siga estas etapas simples para configurar o Tradutor de Legendas SRT e começar a desfrutar de melhores traduções:

### Pré-requisitos

- Uma chave API do Google Gemini (obtenha [aqui](https://aistudio.google.com/), caso ainda não tenha)
- Node.js e npm instalados em sua máquina

### Instalação

1. Clone o repositório: `git clone https://github.com/TiagoStryke/srt-pt-ai`
2. Renomeie `.env.example` para `.env.local` e adicione sua chave API do Google Gemini
3. Instale as dependências usando `npm install`
4. Inicie o servidor localmente com `npm run dev`

Agora você deve conseguir acessar a aplicação em [`http://localhost:3000/`](http://localhost:3000/) no seu navegador.
